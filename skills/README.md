## 📊 技能概览

### 💰 投资分析类
| 技能名称 | 功能描述 |
|---------|---------|
| `akshare-stock` | A股量化数据分析（行情、财务、板块） |
| `china-stock-analysis` | 中港股票分析与投资建议 |
| `stock-monitor-skill` | 智能股票监控预警系统（7大规则） |
| `stock-watcher` | 自选股管理监控 |
| `investment-workflow` | 投资档案管理工作流 |
| `tushare-pro` | A股/港股/期货量化数据 |
| `mx-finance-data` | 东方财富金融数据查询 |
| `mx-finance-search` | 全网财经信息搜索 |
| `mx-macro-data` | 宏观经济数据查询 |
| `mx-stocks-screener` | 多维度股票筛选器 |

## 🚀 快速开始

### 安装 OpenClaw
```bash
npm install -g openclaw
```

### 安装技能
```bash
openclaw skill install skills/stock-monitor-skill
```
